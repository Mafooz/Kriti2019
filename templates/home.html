The E-Course System

This is an Online E-Learning Platform created for students of IIT Guwahati where they can get all the necessary material.

Courses can be accessed by everyone!

For Sharing notes, user has to verify that he/she is student of IIT Guwahati for which an OTP will be sent to user's Outlook Credentials.

User can share notes and books in form of PDF which make this platform more helpful for students.

User can also recommend to include some course which will be done by suitable admin based on recommendation.
